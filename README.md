## Deskripsi Proyek
Proyek ini bertujuan untuk menganalisis dan memprediksi jumlah putus sekolah
di Indonesia berdasarkan indikator pendidikan per provinsi menggunakan
Random Forest Regressor.

## Struktur Folder
- notebook/ : kode analisis data & pemodelan
- frontend/ : aplikasi web (HTML, CSS, JavaScript)
- data/ : dataset yang digunakan

## Model
- Baseline: Ridge Regression
- Model Utama: Random Forest Regressor

## Evaluasi Model
- MAE
- RMSE
- R²
- 5-Fold Cross Validation
